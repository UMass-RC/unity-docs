Unity Cluster Roadmap

The Unity Cluster is currently in alpha phase of development. During the alpha phase, expect unexpected downtime, instability, and general frustration once in a while. Here is our development plans for the remainder of the alpha phase:

General

  • Addition of Java 8,10,11 as environment modules
  • Addition of any other common environment modules requested
  • Containerized jobs (Using lxc/lxd)
  • Support for docker/singularity
  • VNC sessions within JupyterLab for GUI applications
  • MPI setup for jobs running on multiple nodes.

Website

  • About Us Page
  • Finish User Documentation

Hardware

  • Addition of a flash storage node

Once the above are completed and we are able to test many scenarios to make sure the cluster is stable, we will start the public launch. The cluster will then enter beta phase.

In beta phase you should expect expected downtime (emails sent out), with unexpected downtime occuring only on the rarest of occasions (usually a mistake). The cluster should be stable enough to actively use in a research environment. We will work on the following during the beta phase:

  • Proper Slurm Accounting
  • Redundant hardware encase of failures
